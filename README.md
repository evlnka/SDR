# SDR 

**Автор:** Коваленко Евангелина ИА-331

### Занятие 1
**Тема:** Архитектура Adalm Pluto SDR. GNU Radio. Построение радио-приёмника

### Занятие 2  
**Тема:** Знакомство с библиотеками Soapy SDR, Libio для работы с Adalm Pluto SDR. Инициализация SDR-устройства. Работа с буфером: получение цифровых IQ-отсчетов.

### Занятия 3-4
Тема: Примеры формирования I/Q-сэмплов произвольной формы. Работа с буфером приема SDR (Прямоугольный и треугольный сигналы)

### Занятия 5-6
Тема: Имитация аналоговой передачи звука и его прием с использованием SDR. Анализ влияния чувствительности приемника и усиления передатчика на качество принятых отсчетов сигнала (семплов)
